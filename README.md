# **CivicTech Town (シビックテックタウン)**

シビックテック（市民による技術活用）をテーマにした、半協力・半対戦型のカードゲームです。

地域課題（プロジェクト）を「データ」「技術」「対話」のリソースを駆使して解決し、街のハピネスポイント（HP）を稼ぎましょう！

## **📦 内容物**

このリポジトリには、ゲームを遊ぶために必要なすべてのデザインデータが含まれています。

データ管理と印刷のしやすさを考慮し、すべてHTML/CSS(Tailwind)の単一ファイルで構成されています。

* Read Me.md： このファイル  
* gmmanual.html： GM向けのフルセットのマニュアル（A4印刷・Web閲覧用）  
* playermanual.html： プレイヤー向けの簡易マニュアル  
* carddesign.html： 全カードのデザインデータ（A4サイズ印刷用）  
* playmat\_a3x2.html： プレイマットのデザインデータ（A3サイズ2枚貼り合わせ用）

## **🖨️ 遊び方**

各HTMLファイルをブラウザ（Chrome等）で開き、そのまま印刷（またはPDF保存）してください。

厚手のA4用紙などに印刷し、切り取ればすぐに遊ぶことができます。

※ 印刷時の推奨設定：倍率100%、背景のグラフィックをON、両面印刷（長辺綴じ）
